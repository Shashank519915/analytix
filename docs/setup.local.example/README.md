# Private operator notes (template)

Copy to `docs/setup.local/` — that folder is **gitignored** and never pushed.

```bash
mkdir -p docs/setup.local
cp docs/setup.local.example/MY-DEPLOYMENT.md docs/setup.local/MY-DEPLOYMENT.md
```

Fill in your real Netlify URLs, env checklist, and site keys.  
Public docs use placeholders like `https://your-analytix.example.com`.
